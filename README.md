# react-native-toastfx

react-native-toastfx is a lightweight, fully customizable Toast notification library for React Native, supporting smooth animations and visual effects. It works in both Expo managed and React Native CLI apps, providing an easy-to-use global API to show temporary messages, alerts, or status notifications anywhere in your app.

## Installation


```sh
npm install react-native-toastfx
```


## Usage


```js
import { multiply } from 'react-native-toastfx';

// ...

const result = await multiply(3, 7);
```
